## 🛠️ Estructura del Repositorio

El proyecto está organizado de la siguiente manera:

```text
📦 rnaab_riesgo_crediticio
 ┣ 📂 docs                   # Archivos fuente para el despliegue en GitHub Pages
 ┃ ┣ 📂 assets/css           # Estilos personalizados de la página web
 ┃ ┣ 📜 _config.yml          # Archivo de configuración de Jekyll/GitHub Pages
 ┃ ┣ 📜 index.md             # Página principal del reporte
 ┃ ┣ 📜 exploracion.md            # Análisis de exploración de datos
 ┃ ┣ 📜 modelos.md            # Desarrollo de modelos y evaluación
 ┃ ┣ 📜 casodeuso.md            # Implementación de scorecard en casos reales
 ┃ ┗ 📜 sitio-web.md                # Presentación de la solución final
 ┣ 📂 output                 # Recursos generados por el notebook
 ┃ ┣ 📂 figs     # Gráficos de comparación y evaluación de los modelos
 ┃ ┣ 📂 models         # Modelos guardados
 ┃ ┗ 📜 datos_distribucion.json         # Datos de distribución
 ┗ 📜 RNAAB_T2_riesgo_crediticio.ipynb      # Código fuente en Python
```
